# aitaotu-spider
爬取爱套图网图片，https://www.aitaotu.com/
=========================================

运行环境：
--------
* windows7 x64
* python3.7.0

python库：
---------
* requests
* BeautifulSoup
	
设置爬取参数：
------------
* photo_type = 'dmtp'  	  # 资源类型
* pages = 1               # 爬取页数
* path = 'C:\\photos\\'   # 保存路径
